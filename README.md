# SmartSpend AI

An AI-powered Expense Tracker and Personal Finance Management System built using Python, Streamlit, SQLite, Pandas, Plotly, and Machine Learning.

## Features

- Add and manage expenses
- Monthly budget tracking
- Savings analysis
- Expense history management
- Expense prediction using Machine Learning
- Interactive dashboards and visualizations
- Export reports in CSV and PDF formats

## Tech Stack

- Python
- Streamlit
- SQLite
- Pandas
- Plotly
- Scikit-Learn
- ReportLab
